# JAMpol25 LHAPDF files

## Paper
https://inspirehep.net/literature/2935078

## Description
- JAMpol25-PPDF_proton_nlo: Proton helicity PDFs
- JAMpol25-PSTF_proton: Polarized structure functions $g_1$ and $g_2$ (proton)
- JAMpol25-PSTF_neutron: Polarized structure functions $g_1$ and $g_2$ (neutron)
- JAMpol25-PSTF_deuteron: Polarized structure functions $g_1$ and $g_2$ (deuteron)
- JAMpol25-PSTF_helium: Polarized structure functions $g_1$ and $g_2$ (helium)
- JAMpol25-PSTF_tritium: Polarized structure functions $g_1$ and $g_2$ (tritium)
- JAMpol25-PSTF_proton_LT_only: Polarized structure functions $g_1$ and $g_2$ (proton) (LT only)
- JAMpol25-PSTF_neutron_LT_only: Polarized structure functions $g_1$ and $g_2$ (neutron) (LT only)
- JAMpol25-PSTF_deuteron_LT_only: Polarized structure functions $g_1$ and $g_2$ (deuteron) (LT only)
- JAMpol25-PSTF_helium_LT_only: Polarized structure functions $g_1$ and $g_2$ (helium) (LT only)
- JAMpol25-PSTF_tritium_LT_only: Polarized structure functions $g_1$ and $g_2$ (tritium) (LT only)
