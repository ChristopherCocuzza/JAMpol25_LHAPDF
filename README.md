# JAMpol25 LHAPDF files

## Paper
https://inspirehep.net/literature/2935078

## Description
- JAMpol25-PPDF_proton_nlo: Proton helicity PDFs
- JAMpol25-PSTF_proton: Polarized structure function (proton)
- JAMpol25-PSTF_neutron: Polarized structure function (neutron)
- JAMpol25-PSTF_deuteron: Polarized structure function (deuteron)
- JAMpol25-PSTF_helium: Polarized structure function (helium)
- JAMpol25-PSTF_tritium: Polarized structure function (tritium)
- JAMpol25-PSTF_proton_LT_only: Polarized structure function (proton) (LT only)
- JAMpol25-PSTF_neutron_LT_only: Polarized structure function (neutron) (LT only)
- JAMpol25-PSTF_deuteron_LT_only: Polarized structure function (deuteron) (LT only)
- JAMpol25-PSTF_helium_LT_only: Polarized structure function (helium) (LT only)
- JAMpol25-PSTF_tritium_LT_only: Polarized structure function (tritium) (LT only)
